CSS Layouts

11.10.2020

Flexbox:

Gjennom Flexbox-funksjonen kan du spesifisere en container gjennom displayet: flex-egenskap, så kan du fleksibelt manipulere alle elementene i denne beholderen, for eksempel å bestemme formen på utseendet til elementene (vannrett eller loddrett), bestemme størrelsen på hvert element og endre posisjonen til elementene enkelt.

Flexbox kan brukes i nesten alt. Du kan for eksempel bruke den i Navbar for å bygge en toppmeny, enten det er vannrett eller loddrett.

Grid:

Grid fungerer i større skala enn Flexbox, ettersom Flexbox bare kontrollerer elementene inni den og bestemmer deres posisjonering, mens Grid hovedmål er å dele siden helt for å lette responsen under utvikling. I Grid definerer vi området for hver del, ved å dele siden til 10 deler, for eksempel, gir vi seks deler for innholdet på siden, og kanskje fire deler for sidefeltet og så videre.

Grid kan brukes på hele delen av siden, eller på delene av siden, eller til og med på mini-delene på siden uten problemer, da Grid setter de nødvendige grensene for hvert element som ikke vil overskride det så lenge alle grenser er spesifisert for det når det gjelder lengde, bredde, kanter og avstand.
